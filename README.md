<h2 align="center" id="Inspiration"> :label: Inspiration</h2>

Considering the importance of safety and security of women in India, I wanted to give it a try from my end to address the issue in a simpler and safer way.  The app is about safety as well as awareness & makes the user safe & confident. It is easy to use because of it's user friendly and easy to navigate UI. I wanted to give them a tap on the shoulder to keep going and not give up! 
<br>
Keeping this mind I developed an application named <b>GoSecure</b>.


<br>

<h2 align="center" id="Features"> :label: Features</h2>

  
   :point_right: SOS - SOS call and SOS messages to user provided contacts<br/>
   :point_right: Shake - Detects the frequency of shakes and after a certain frequency sends help message with user location to provided contacts<br/>
   :point_right: SOS Pin - User can set up a pin which they can enter while sending a false alert to authenticate that user herself is sending the false alert <br/>
   :point_right: Police stations - Locates all the nearest police stations<br/>
   :point_right: Hospitals - Locates all the nearest hospitals<br/>
   :point_right: Taxi - One touch ola and uber cab facility to books cabs for user<br/>
   :point_right: Articles - To guide the users about self defense techniques and some inspiring stories<br/>
   :point_right: LiveSafe - To send alert to 1 selected contact every 15mins about user's current location<br/>
   :point_right: Low battery alert - To alert user on their low battery which can cause denial of service by app<br/>
